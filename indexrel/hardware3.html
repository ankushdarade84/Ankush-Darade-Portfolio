Mini Oscilloscope and Function Generator

For the Portable Oscilloscope and Function Generator Using Arduino,We used a minimal number of components and circuitry for the development of our project as compared to the existing system. This system consists of Arduino nano, Oled display, voltage divider ckt, Ic Ad9833.With the help of this project we can Measure the signal and see the waveform of signals, also we can generate mutipal signal like sine, square, triangular. with minimum number of circuit.due to less no. of components it is portable device.
