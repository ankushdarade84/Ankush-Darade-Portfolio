Smart Storage System for Onion

We connected the DHT 11 sensor for to read the Temperature and Humidity it read the values and send to controller it checks the value. If temp is greater than desired value then Fan turn On. If Humidity is Greater than 80 then the Heater and Fan turn on. Hence it will maintain the atmosphere. The MQ135 Gas Sensor is sense the gas if gas found (rottening onion smell) found it turn on Buzzer. So, we will separate the good onion from rotten onions.
