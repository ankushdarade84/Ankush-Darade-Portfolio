"Mobile Phone Reparing Center"

"Our web-based application facilitates mobile device query submissions, technician allocation, and quotation management. Developed using Java and React, with backend data storage in MySQL, it ensures secure encrypted passwords and efficient ticket management. Technicians can toggle their availability, ensuring seamless ticket distribution, while filter modes enhance user experience."
